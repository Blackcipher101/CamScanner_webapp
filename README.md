## CamScan
This a lightweight tool for making documents on the fly from camera images.
So this uses OpenCV and Django in tandom also pillow is used to make the pdf

### Features
1)To PDF<br>
2)Image Sharpen<br>
3)Document like look<br>
4)Multipagepdf<br>
5)Cropping<br>
6)Test your Crop<br>
